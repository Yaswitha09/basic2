SGPA Based Speed Control of an Induction Motor

This project aims to design and implementation of a Variable Frequency Controller based on the Pulse Width Modulation (PWM) Technique for single-phase and three Phase Induction motors using VHDL. Induction motors are widely used in industrial sectors as actuators. These motors are used because of their low cost and robustness in nature. Induction motors are widely used to control the speed of conveyor systems, blower speeds, pump speeds, and many other applications. This project is about FPGA based speed control IC for three-phase induction motor drives. The sinusoidal PWM is realized on a single FPGA chip from Xilinx Inc. speed control is done by adjusting the modulation index and frequency from the FPGA side. Speed control has been developed using Xilinx software using VHDL. This provides a wide range of controlling speed in an induction motor
